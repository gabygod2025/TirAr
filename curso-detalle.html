SOBRE EL CURSO

Formación Integral

Nuestro curso de idoneidad está diseñado para cumplir con todos los requisitos que se piden para obtener el CLU, también para hombres y mujeres que desean iniciarse en la práctica de tiro, ya sea con fines defensivo o deportivos y más importante aún, para formarse como un tirador responsable y seguro. No solo te preparamos para el examen, te enseñamos a disparar de forma segura, a respetar, a dominar el uso y conservación de armas. Además se puede usar como una jornada para quienes quieren acercarse al mundo de las armas de manera recreativa, disfrutando de disparar con diferentes calibres bajo la supervisión de instructores certificados. Ideal para regalar, compartir en grupo o dar tu primer paso sin la necesidad de iniciar el trámite del CLU.

PREGUNTAS FRECUENTES

¿Qué requisitos necesito para hacer el curso?

Ser mayor de 18 años, DNI vigente y no tener antecedentes penales. Nosotros te asesoramos con el resto de la documentación.

¿Necesito tener arma propia?

No. La academia provee todo el equipamiento necesario: armas, municiones, protección auditiva y visual.

¿Cuánto dura el certificado de idoneidad?

El certificado tiene validez para presentar el trámite ante la ANMaC. Una vez obtenida la CLU, esta tiene una vigencia de 5 años.

¿Dónde se realizan las prácticas?

Las prácticas se realizan en polígonos habilitados en Buenos Aires, bajo estricta supervisión de instructores certificados.
